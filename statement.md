Promblem statement:
							This project contains a list of cars which are economical and 
              expensive and the users can search for a car based on their need
              using different factors which can be convenient for them.

Scope of the project:
							This project aims to provide a suitable car which the user is looking for.

Target users:
							It targets the users who are going to buy a new car and those who don't know much about the cars

High-level features:
						 It helps in searching cars based on different factors like ( specific car , cars launched in a specific year , cars within a price range , cars within a year range , cars of a certian brand)
             and it provides you the link of the specific car so that you can know more.
